{% elif idp.slug == "citizensafe" %}

CitizenSafe - citizen identity verification for the public sector.

CitizenSafe allows you to access and transact with online government services, rather than through traditional contact channels. Data security is critical, and putting personal information and entitlements online carries a risk. CitizenSafe delivers a single and secure interface that verifies you when you first use the system, then issues a unique electronic identity ‘token’ for all subsequent online transactions.

Features of CitizenSafe:

  • instant verification
  • one username and password for all online government services
  • eliminates the inconvenient and inefficient traditional practices for verifying user identity

CitizenSafe is part of CitizenSafe.
